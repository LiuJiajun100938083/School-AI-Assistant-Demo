# 校园AI助手使用指南

## 文件上传功能

### 支持的格式
- **.txt** - 纯文本文件
- **.docx** - Microsoft Word文档
- **.pdf** - PDF文档
- **.pptx** - PowerPoint演示文稿
- **.md** - Markdown文档

### 使用步骤
1. 点击📎按钮打开文件上传界面
2. 选择或拖拽文件到上传区域
3. 等待文件处理完成
4. 选择要用于对话的文件
5. 开始基于文档的智能对话

### 注意事项
- 单个文件最大10MB
- 建议上传清晰的文档
- 支持多个文件同时使用

## 学习建议
使用文档上传功能可以：
- 上传课程资料进行讨论
- 分析学习文档内容
- 基于教材提问
- 整理学习笔记
